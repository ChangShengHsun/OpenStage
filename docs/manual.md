# GridStage 使用說明書

GridStage 是開源免費的舞台隊形編輯器：把每一個隊形畫在格線舞台上，配上音樂與
時間軸，就能播放整場演出的走位動畫，並匯出成 PDF、影片、GIF 給團隊。
本說明書涵蓋所有功能；App 內的「教學」按鈕有帶截圖的入門版。

> 名詞：**隊形（formation）**＝某個時間點所有人的站位快照；**走位（transition）**
> ＝兩個隊形之間的移動；**播放頭（playhead）**＝時間軸上目前的時間點。

## 1. 安裝

### 1.1 Windows / macOS 桌面版（一般使用者）

[Releases](https://github.com/ChangShengHsun/GridStage/releases) 下載
`GridStage Setup <版本>.exe`（Windows）或 `.pkg`（macOS）。未簽章：Windows
SmartScreen 按「其他資訊 → 仍要執行」；macOS 對安裝檔按右鍵 → 打開。
桌面版完全離線可用，Windows 版會自動更新。

### 1.2 手機／平板（Android / iOS）

瀏覽器開 <https://changshenghsun.github.io/GridStage/>，
Android Chrome：選單 ⋮ → 加到主畫面；iOS Safari：分享 → 加入主畫面。
之後從主畫面開啟＝全螢幕 App，第一次載入後離線可用。

### 1.3 從原始碼（開發者）

Node.js 20+ 與 pnpm：`pnpm install && pnpm dev:web`。
完整自架（含協作伺服器與後端）：`docker compose up`（見 README）。

## 2. 介面總覽

- **上欄**：標題、作品庫、復原／重做、即時共編、匯出、偏好設定、教學。
- **左欄（演出者）**：名單、分組、班底、道具。
- **中央舞台**：拖曳舞者標記排位；右上角有吸附／畫筆／便籤／3D 切換。
- **右欄（屬性）**：選到誰就顯示誰——舞者、隊形或道具的細節。
- **下欄（時間軸）**：隊形卡片、節拍、播放控制。
- 手機窄螢幕時左右欄收成畫面兩側的「演出者」「道具」抽屜標籤。
- **簡易／完整模式**：預設簡易，只顯示核心功能；⚙ 偏好設定切到「完整」
  顯示全部進階工具（下述標 ⚙ 的功能都在完整模式）。

## 3. 建立舞碼與舞台設定

- **作品庫**：新增／切換／複製／封存／刪除多支編舞；新增時可選舞台尺寸
  預設（鏡框式 12×8、教室、黑盒、體育館、自訂）。
- **舞台設定**：長寬（公尺）、觀眾席方位（bottom＝導演視角；top＝表演者
  視角，畫面翻轉 180°）、側台／後台暫留區、場地背景照片（含透明度）。

## 4. 演出者名單

- 新增舞者、姓名／角色／顏色；**匯入 CSV**（欄位：姓名、角色、顏色）。
- **本次上場**：取消勾選＝替補或請假——舞者保留所有站位，但不出現在
  舞台、匯出與分析；名單中以灰字加「缺席」顯示，隨時勾回。
- ⚙ **標示（badge）**：一個中文字或 ≤4 英文字母，顯示在標記圈內與 3D 臉上。
- ⚙ **分組**：逗號分隔的組名（前排、空翻組…），名單上點組名整組選取。
- ⚙ **班底（crews）**：把整份名單存起來，其他編舞一鍵載入。
- ⚙ **道具**：矩形／圓／三角形景片，跟舞者一樣有每隊形位置。

## 5. 排隊形

- 拖曳標記；**框選**空白處拉出選取框；拖任一已選標記＝整組移動。
- **吸附**：0.5m 格點；**對齊成排／成列、水平／前後均分**；**交換兩人**；
  **左右鏡射**；⚙ **群組旋轉 ±15°、散開／收攏**（形狀不變）。
- **範本**：直線、V 形、圓、網格一鍵排開；⚙ **我的隊形庫**：存自己的
  形狀跨作品重用；⚙ **建議隊形**：依走動距離、間距、對稱自動提三案。
- **解交叉（Untangle）**：重新指派誰站哪，讓上一隊形走過來不打結；
  走位若會相撞或必須用跑的，隊形面板會出現警告。
- 每人**面向**（0°＝面對觀眾）與 ⚙ **狀態記號**（方／三角／菱形圈，
  意義由團隊自訂，例如蹲下、跳躍）。
- ⚙ **畫筆／便籤**：在隊形上手繪註記或釘文字，會印進 PDF 隊形圖。

## 6. 時間軸

- **隊形卡片**：拖曳移動時間、拉左右邊緣改停留長度；點選即編輯。
- 隊形欄位：**開始（秒）／停留（秒）／走位（秒）**——走位＝從上一個
  隊形走過來的時間，改它只移動這個隊形，後面的隊形不動。
- ⚙ **卡片顏色**：一個歌曲段落一個顏色，時間軸一目了然。
- **段落標記**：主歌、副歌等文字旗標。
- **循環**：按下後時間軸出現藍色區間與兩根可拖曳的竿子（預設播放頭
  前後各 4 秒），播放在區間內反覆；竿子靠近隊形邊界會自動吸附。
- ⚙ **播放速度** 0.5–2×；**節拍器**（有 BPM 時）。

## 7. 音樂與節拍

- **上傳音訊**後播放以音樂為時鐘；波形下可打**節拍標記**。
  已載入參考影片時聲音以影片為準，上傳音訊按鈕會暫時鎖住。
- **節拍… 對話框**：BPM、打拍測速（tap tempo）、**計數段落**——8 拍計數
  (8ct) 從每段的第 1 拍重新起算，尺標與時間碼都會顯示。

## 8. 參考影片與 AI 隊形擷取 ⚙

- **Ref video**：載入排練／比賽影片，浮窗或分割畫面與時間軸同步播放
  （影片聲音為主時鐘）；「對齊到播放頭」一鍵校準時間偏移。浮窗可由
  標題列拖曳、右下角把手調整大小；分割模式的分隔線也能拖曳調比例。
- **校準舞台**：把 4 支釘子拖到影片中舞台四角，綠色公尺網格貼合地板
  即正確。校準時影片會縮小置中，釘子可以拖到畫面外——攝影機沒拍到的
  舞台角落也能標；標題列的 ⛶ 可把整個視窗放大到全螢幕慢慢校。
- **擷取隊形**：暫停任一畫面，AI 偵測舞者位置＋面向，直接填進目前
  隊形（不確定的人會被選取待你確認）。
- **掃描整支影片**：每秒取樣，用每個「停住的隊形」**重建整份隊形圖**
  （取代前會先跳確認視窗），過場不會誤加；一次復原可找回原本的。
  全程在你的裝置上運算，不上傳。第一次使用擷取／掃描會下載約 200 MB
  的偵測模型（下載一次後快取）。
- **檢討**：把暫停畫面與播放頭當下的隊形圖比對——每人偏差多少（偏左右
  ／前後）、C位有沒有站在中線上（先選取一位舞者＝指定C位）、左右對稱度
  （0＝完全對稱）。排練影片一格一格檢討最好用。

## 9. 3D 預覽

右上角「3D」切換：立體舞台＋攝影機預設（觀眾席／俯視／側面）、
**跟隨某位舞者**的追蹤鏡頭。僅檢視，編輯回 2D。

## 10. 即時協作

- **即時共編**：產生連結傳給隊友，多人同時編輯，游標與選取即時可見；
  每人有自己的復原鍵。**檢視連結**＝唯讀版。
- 點隊友的顏色圓點＝**跟隨**對方的播放頭與選取。
- **留言**：對整個隊形或單一舞者留言；**✓ 標記已解決**收進「已解決」
  摺疊區，↺ 可重新開啟。

## 11. 匯出與備份

| 格式              | 內容                                                                                   |
| ----------------- | -------------------------------------------------------------------------------------- |
| 影片              | 實時錄製整場（2D 或 3D）；可夾帶參考影片（右上小視窗或並排），有參考影片時聲音用影片的 |
| GIF               | 整場動圖（640px／8fps），離線算圖，適合丟群組                                          |
| PDF · 隊形圖      | 每隊形一頁的可列印舞台圖＋名單頁（含註記，支援中文）                                   |
| PDF · 個人走位單  | 每位舞者一頁：路線、座標、時間與 8 拍數                                                |
| PDF · 排練交接包  | 名單＋所有隊形圖＋所有個人單合訂                                                       |
| PNG               | 目前隊形的單張快照                                                                     |
| 檔案 (.gridstage) | 整份編舞——備份或傳給別人（音樂／背景圖不含）                                           |

作品存在**該裝置的瀏覽器**裡；清除瀏覽資料會刪掉它。系統累積一定
內容後每週提醒你匯出 .gridstage 備份；換裝置＝匯出檔案→對方匯入
（舊版 .gridstage.json 也能匯入）。

檔案怎麼傳最快：

- **桌面版**：隊友傳來的 .gridstage 檔點兩下，直接用 GridStage 開啟。
- **手機（Android，已加入主畫面）**：收到檔案後「分享 → GridStage」
  即匯入；匯出畫面的「分享…」鈕則直接喚起系統分享面板往外傳。
- **iPhone**：先存到「檔案」，再用「文件庫 → 匯入檔案」開啟
  （iOS 不支援分享進 PWA）。

## 13. 常見問題

- **Windows 說這個 App 有風險？** 未購買簽章憑證所致；「其他資訊 →
  仍要執行」。原始碼公開可查。
- **作品不見了？** 多半是瀏覽器資料被清除。平時用「匯出 → 檔案」留
  .gridstage 備份即可還原；桌面版不受瀏覽器清理影響。
- **影片解不開？** 用 MP4（H.264）最保險。
- **手機上排位不準？** 用兩指縮放頁面前先確認沒開啟瀏覽器縮放；
  App 內拖曳本身支援觸控。
- **AI 擷取抓錯人？** 擷取後不確定者會自動被選取，拖到正確位置即可；
  它是草稿工具，最後一步永遠是你的眼睛。
